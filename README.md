# Neural Day Trader Platform

Plataforma SaaS de trading quantitativo automatizado com React, TypeScript, Supabase e IA/ML.

## Recursos Principais

- ✅ 300+ ativos (Forex, Cripto, Índices, Commodities, Ações)
- ✅ Integração MetaAPI para MT5
- ✅ Sistema de IA preditiva com análise neural
- ✅ Assistente de voz Luna
- ✅ Social Intelligence (Twitter, Reddit, Telegram)
- ✅ Sistema completo de gráficos e análise técnica
- ✅ Gestão de risco avançada

## Tecnologias

- React 18.3 + TypeScript
- Tailwind CSS v4
- Supabase para backend
- Vite para build
- MetaAPI Cloud SDK

## Como usar

1. Faça login na Landing Page
2. Conecte sua conta MetaTrader 5
3. Configure suas estratégias de trading
4. Ative o modo automático ou use análises manuais
